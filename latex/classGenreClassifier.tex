\doxysubsection{Detailed Description}
Performs automatic genre classifications based on incoming audio data. 

Uses the \mbox{\hyperlink{classAudioCapture}{Audio\+Capture}} class to get audio, then uses \mbox{\hyperlink{classGenrePredictor}{Genre\+Predictor}} class to make requests to a webserver with WAV-\/formatted audio and receive regular predictions, which are passed to callback function.

Detailed Description

Performs automatic genre classifications based on incoming audio data.

Uses the AudioCapture class to get audio, then uses GenrePredictor class to make requests to a webserver with WAV-formatted audio and receive regular predictions, which are passed to callback function.

!Warning: Singleton class !Do not attempt to instantiate multiple instances of this class

Callback function for predicting genre of incoming audio.

This function is passed to AudioCapture class, which calls it with available audio data. It accumulates audio until it has the required length and makes a prediction on the data. Lastly, prediction results are passed onto provided callback function 'callback'

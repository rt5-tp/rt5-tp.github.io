Detailed Description

Special buffer calls callback when full.

Fills up one of two buffers over time through 'add_data' calls and makes call to callback with the full buffer.

void PingPongBuffer::add_data ( const std::vector< short > &  data)

Function for adding data to the PingPongBuffer.

Fills the current buffer with the provided data. If buffer reaches capacity, all data contained in it is sent out via callbacks, then buffer is switched out and remaining data added to new buffer.
